# Navigation

Go Top
------
Normal mode:

```
gg
```

Go Bottom
---------
Normal mode:
```
G
``` 
Open a Source File
------------------
Place cursor on `stdio.h` and in normal mode "go  forward":

```
gf
```

To return, `ctrl O`.

Open in new tab:

```
ctrl-W
gf
```
Close newly opened tab:

```
:bd
```

Open man Page
-------------
Cursor on a command:

```
shift k
```

Open Terminal within Vim
------------------------
```
Ctrl z
```

Return to vim:

```
fg
```
